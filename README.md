<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

------
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation



1. Clone the repo
   ```sh
   git clone MY ADDRESS
   ```
2. Composer
   ```sh
   composer install
   ```
3. .env Configration
   ```
   Rename or copy .env.example file to .env
   ```
4. Create Key
   ```
    php artisan key:generate
   ```
5. Set your database credentials in your .env file
